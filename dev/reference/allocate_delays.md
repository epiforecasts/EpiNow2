# Allocate Delays into Required Stan Format

**\[stable\]** Allocate delays for stan. Used in
[`delay_opts()`](https://epiforecasts.io/EpiNow2/dev/reference/delay_opts.md).

## Usage

``` r
allocate_delays(delay_var, no_delays)
```

## Arguments

- delay_var:

  List of numeric delays

- no_delays:

  Numeric, number of delays

## Value

A numeric array
