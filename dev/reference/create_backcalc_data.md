# Create Back Calculation Data

**\[stable\]** Takes the output of
[`backcalc_opts()`](https://epiforecasts.io/EpiNow2/dev/reference/backcalc_opts.md)
and converts it into a list understood by stan.

## Usage

``` r
create_backcalc_data(backcalc = backcalc_opts())
```

## Arguments

- backcalc:

  A list of options as generated by
  [`backcalc_opts()`](https://epiforecasts.io/EpiNow2/dev/reference/backcalc_opts.md)
  to define the back calculation. Defaults to
  [`backcalc_opts()`](https://epiforecasts.io/EpiNow2/dev/reference/backcalc_opts.md).

## Value

A list of settings defining the Gaussian process
