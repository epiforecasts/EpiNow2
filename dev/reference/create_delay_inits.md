# Create initial conditions for delays

Create initial conditions for delays

## Usage

``` r
create_delay_inits(stan_data)
```

## Arguments

- stan_data:

  A list of data as produced by
  [`create_stan_data()`](https://epiforecasts.io/EpiNow2/dev/reference/create_stan_data.md).

## Value

A list of initial conditions for delays
