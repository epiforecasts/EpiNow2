# Get Folders with Results

**\[stable\]**

## Usage

``` r
get_regions(results_dir)
```

## Arguments

- results_dir:

  A character string giving the directory in which results are stored
  (as produced by
  [`regional_epinow()`](https://epiforecasts.io/EpiNow2/dev/reference/regional_epinow.md)).

## Value

A named character vector containing the results to plot.
