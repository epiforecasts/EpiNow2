# Create parameters for stan

Create parameters for stan

## Usage

``` r
create_stan_params(params)
```

## Arguments

- params:

  A list of `<EpiNow2.params>` as created by
  [`make_param()`](https://epiforecasts.io/EpiNow2/dev/reference/make_param.md)

## Value

A list of variables as expected by the stan model
