# Validate probability distribution for passing to stan

**\[stable\]** `check_stan_delay()` checks that the supplied data is a
`<dist_spec>`, that it is a supported distribution, and that is has a
finite maximum.

## Usage

``` r
check_stan_delay(dist)
```

## Arguments

- dist:

  A `dist_spec` object.\`

## Value

Called for its side effects.
